===== About =====

=== Description ===

Alinea.Caribu allows to compute light distribution on a set of triangles. 
  It combines a projection algorithm for direct lighting and the nested radiosity 
  algorithm for multiple rediffusion. 
  It can simulate an infinite canopy around your scene. 
  The package is based on four C binaries (Canestra, s2v, MCsail and periodise) 
  which are combined in a python script (caribu). 
  Visualisation of outputs is available through PlantGL.
